# WebMall backup data

This directory holds the backup tarballs for the 4 WooCommerce shops.

Required files (downloaded automatically by `scripts/setup_webmall.sh`):

| File | Content | Size |
|------|---------|------|
| `mariadb_data_shop1.tar.gz` | MariaDB data for shop 1 | ~21 MB |
| `mariadb_data_shop2.tar.gz` | MariaDB data for shop 2 | ~27 MB |
| `mariadb_data_shop3.tar.gz` | MariaDB data for shop 3 | ~26 MB |
| `mariadb_data_shop4.tar.gz` | MariaDB data for shop 4 | ~26 MB |
| `wordpress_data_shop1.tar.gz` | WordPress data for shop 1 | ~847 MB |
| `wordpress_data_shop2.tar.gz` | WordPress data for shop 2 | ~1.1 GB |
| `wordpress_data_shop3.tar.gz` | WordPress data for shop 3 | ~850 MB |
| `wordpress_data_shop4.tar.gz` | WordPress data for shop 4 | ~688 MB |

Source: `data.dws.informatik.uni-mannheim.de/webmall/backup/`
